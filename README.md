# Secret Diary

A personal journaling web application built with Flask.

This project is a rebuilt and improved version of my original Secret Diary application. The goal of this version is to follow a more professional Flask architecture using Blueprints, SQLAlchemy, Flask-Login, Flask-Migrate, MySQL, and Bootstrap.

## Features

* User Registration & Login
* Secure Authentication
* Create Diary Entries
* View Personal Entries
* Edit Entries
* Delete Entries
* User Profile Page
* Responsive Bootstrap Interface

## Tech Stack

* Python
* Flask
* SQLAlchemy
* Flask-Login
* Flask-Bcrypt
* Flask-Migrate
* MySQL
* Bootstrap 5

## Project Structure

This project follows a modular Flask architecture using Blueprints:

* Auth Blueprint
* Diary Blueprint
* Profile Blueprint

Additional features and improvements will be added as the project grows.
